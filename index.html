Druid介绍.

需要选择Druid的理由? 不要再观望.

为社区, 为nerds.

由阿里巴巴的温少开发, Druid 基于JDK 1.6，和JDBC 4.0开发, 使用 Maven 管理依赖, 使用 GitHub 管理版本，帮助贡献在开发更棒的RDBMS数据库相关程序。

每个人都适用.

Druid被设计为JDBC组建库，快速、强劲的JDBC连接池，而且可以监控使用Druid的JDBC连接池。

提供的主要功能.

JDBC连接池，JDBC连接池监视，DruidDataSource, DruidProxyDriver, SQL Parser, 监视的JSON接口等等.

许可条款.

Druid专注点.

专注，别分心.

更快更好的提升RDBMS数据库相关程序的连接池效率，更准确的观察和维护程序.
